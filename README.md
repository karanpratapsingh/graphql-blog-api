## GraphQL Blog API:

### A GraphQl API like Medium.com to query, mutate and subscribe articles along with comments and various authentication features.

## Technologies used:
<ul>
    <li>GraphQl</li>
    <li>Prisma</li>
    <li>Heroku</li>
    <li>ES7</li>
    <li>Node JS</li>
    <li>JWT Auth</li>
    <li>Postgres/Mongo DB</li>
</ul>
